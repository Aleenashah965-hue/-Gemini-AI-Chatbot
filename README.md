# 🤖 Gemini AI Chatbot
🤖 AI Chatbot built with Python, Streamlit, and Google Gemini API featuring multi-language support, customizable settings, and a ChatGPT-style interface.

## ✨ Features

- 🤖 Powered by Google Gemini AI
- 💬 ChatGPT-style Chat Interface
- 🌍 Multi-language Support (English, Urdu, Roman Urdu, Pashto)
- ⚙️ Sidebar Settings Panel
- 🎨 Dark & Light Theme
- 🤖 AI Model Selection
- 😊 AI Tone Selection
- 📏 Response Length Control
- 💾 Chat History
- 📥 Download Chat as TXT
- ❌ Error Handling

---

## 🛠 Technologies Used

- Python 3.x
- Streamlit
- Google Gemini API (`google-genai`)
- python-dotenv

bash
https://github.com/Aleenashah965-hue/-Gemini-AI-Chatbot
### 2. Install Dependencies
bash
pip install -r requirements.txt
### 3. Create a `.env` File
env
GOOGLE_API_KEY=api_key
### 4. Run the Application
bash
streamlit run app.py
